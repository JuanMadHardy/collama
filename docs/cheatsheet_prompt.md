# üìù Perfect Prompt Cheatsheet

## üèóÔ∏è Prompt Scaffold Structure

### 1. **ROLE/AUTHORITY** (Who)
```
Act as a [expert role] with [specific expertise]
Example: "Act as a senior marketing strategist with 15 years experience in digital campaigns"
```

### 2. **GOAL/OBJECTIVE** (What)
```
Your task is to [specific action] [target audience/outcome]
Example: "Your task is to create a compelling product description for eco-conscious consumers"
```

### 3. **CONTEXT/BACKGROUND** (Why)
```
Given that [situation/context], I need you to...
Example: "Given that our target market is shifting toward sustainability, I need you to..."
```

### 4. **SPECIFICATIONS** (How)
```
Follow these guidelines:
- Format: [specific format]
- Length: [word/sentence limit]
- Tone: [professional/casual/etc.]
- Style: [specific style requirements]
```

### 5. **CONSTRAINTS** (Boundaries)
```
Do NOT:
- [Limitation 1]
- [Limitation 2]
Must include:
- [Requirement 1]
- [Requirement 2]
```

### 6. **OUTPUT FORMAT** (Structure)
```
Provide your response in this exact format:
[Section headers, bullet points, numbering, etc.]
```

## üéØ Essential Prompt Elements

### ‚úÖ MUST INCLUDE:
- **Clear objective** - What exactly do you want?
- **Specific context** - Background information
- **Defined audience** - Who is this for?
- **Output format** - How should it be structured?
- **Length parameters** - Word count or size limits
- **Tone/style guidance** - Professional? Casual? Technical?

- [ ] Includes verification steps

## üõ†Ô∏è Common Prompt Patterns

### **Analysis Pattern:**
```
Analyze [SUBJECT] for [AUDIENCE]. 
Consider [FACTORS] and provide [TYPE OF INSIGHT].
Format: [STRUCTURE REQUIREMENTS]
```

### **Creation Pattern:**
```
Create [CONTENT TYPE] for [TARGET AUDIENCE].
Purpose: [SPECIFIC GOAL]
Constraints: [LIMITATIONS]
Include: [REQUIRED ELEMENTS]
Style: [TONE/APPROACH]
```

### **Problem-Solving Pattern:**
```
As [EXPERT ROLE], solve [SPECIFIC PROBLEM] for [STAKEHOLDER].
Current situation: [CONTEXT]
Constraints: [LIMITATIONS]
Deliverables: [EXPECTED OUTPUTS]
Evaluation criteria: [SUCCESS MEASURES]
```

---

## üö´ Common Mistakes to Avoid

‚ùå **Vague instructions**: "Make it better"
‚úÖ **Specific requests**: "Reduce word count by 20% while maintaining key points"

‚ùå **No context**: "Write about AI"
‚úÖ **Full context**: "Write a 500-word article about AI applications in healthcare for hospital administrators"

‚ùå **Missing constraints**: No length or format limits
‚úÖ **Clear boundaries**: "Maximum 300 words, bullet-point format, avoid technical jargon"

---

## üìà Pro Tips

### **Chain Prompts Together:**
1. First prompt: Research and outline
2. Second prompt: Expand on outline
3. Third prompt: Polish and optimize

### **Use Temperature Settings:**
- **0.1-0.3**: Factual, consistent responses
- **0.4-0.7**: Balanced creativity and accuracy  
- **0.8-1.0**: Highly creative, unpredictable

### **Iterative Refinement:**
Start broad ‚Üí Get feedback ‚Üí Add constraints ‚Üí Refine further

---

*Remember: Great prompts = Clear communication + Specific requirements + Defined boundaries*
